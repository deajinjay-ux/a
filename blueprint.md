# DCOING - 음식 고르기 프로젝트

## 개요

이 프로젝트는 사용자가 무엇을 먹을지 결정하기 어려울 때, 재미있고 시각적으로 매력적인 방식으로 음식을 추천해주는 웹 애플리케이션입니다.

## 디자인 및 기능

-   **모던하고 대담한 디자인:**
    -   생동감 넘치는 색상 팔레트와 부드러운 그라데이션을 사용합니다.
    -   입체감을 주는 다중 레이어 그림자 효과를 적용합니다.
    -   미묘한 배경 텍스처로 고급스러운 느낌을 줍니다.
    -   사용자의 이해를 돕는 직관적인 아이콘을 사용합니다.

-   **핵심 기능:**
    -   **음식 추천:** "오늘 뭐 먹지?" 버튼을 클릭하면, 미리 정의된 음식 목록에서 무작위로 하나의 음식을 선택하여 화면에 표시합니다.
    -   **재사용 가능한 컴포넌트:** Web Components 기술을 사용하여 음식 추천 UI를 `food-suggestion`이라는 독립적인 컴포넌트로 만듭니다.
    -   **인터랙티브한 경험:** 버튼 클릭 시 부드러운 애니메이션 효과를 추가하여 사용자의 즐거움을 더합니다.

-   **기술 스택:**
    -   HTML, CSS, JavaScript (프레임워크 없음)
    -   ES Modules를 사용한 모듈화
    -   Web Components (Custom Elements, Shadow DOM)
    -   최신 CSS 기능 (Container Queries, Cascade Layers, :has() 선택자 등)

## 현재 요청 계획

1.  **`blueprint.md` 생성 (완료):** 프로젝트의 목적, 디자인, 기능 및 현재 작업 계획을 문서화합니다.
2.  **HTML 구조 설정:** `index.html`에 웹 컴포넌트를 위한 기본 구조와 "오늘 뭐 먹지?" 버튼을 추가합니다.
3.  **CSS 스타일링:** `style.css`에 전체적인 레이아웃, 색상, 폰트 및 디자인 가이드라인에 따른 스타일을 적용합니다.
4.  **JavaScript 로직 구현:**
    -   `main.js`에서 음식 목록(배열)을 정의합니다.
    -   `food-suggestion` 커스텀 엘리먼트를 생성합니다.
    -   버튼 클릭 시, 음식 목록에서 무작위로 음식을 선택하고, 애니메이션과 함께 `food-suggestion` 컴포넌트에 표시하는 로직을 구현합니다.
5.  **디자인 개선:** 최종적으로 애니메이션, 그림자, 색상 등 시각적 요소를 다듬어 완성도를 높입니다.
